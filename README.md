* Firebase의 Observe 기능을 이용해 대화 주체 간 실시간으로 데이터를 동기화하여 UI상에 보여지도록 했습니다.

## Teck Stack
* `Firebase` - 백엔드 구성
* `Firestore && Realtime database` - 데이터 저장 및 실시간 동기화
* `SnapKit` - 제약조건 설정
* `RxSwift` - form validation & UI 업데이트
